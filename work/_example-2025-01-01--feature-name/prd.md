# PRD: Example Feature

**Это пример структуры фичи.**

Когда создаёшь новую фичу, она будет в папке:
```
work/YYYY-MM-DD--feature-name/
├── prd.md          # PRD фичи
├── plan.md         # План реализации
└── tasks/          # Задачи
    ├── task-001.md
    ├── task-002.md
    └── ...
```

## Формат названия папки

`YYYY-MM-DD--краткое-название`

Примеры:
- `2025-01-15--user-auth`
- `2025-01-20--export-data`
- `2025-02-01--notifications`

## Используй шаблоны

Шаблоны находятся в `docs/templates/`:
- `prd-template.md` - для ПРД
- `task-template.md` - для задач

## Или используй /brainstorm

Команда `/brainstorm` автоматически создаст всю структуру:
- Задаст вопросы
- Создаст ПРД
- Создаст план
- Разобьёт на задачи

---

Этот пример можно удалить когда создашь первую реальную фичу.
