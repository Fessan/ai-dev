# Инструкции для Codex CLI

Этот файл читается Codex CLI автоматически при старте сессии в репозитории.

## Обязательный порядок

1. Прочитай `INSTRUCTIONS.md` полностью.
2. Определи тип запроса и прочитай соответствующего агента:
   - разработка: `.claude/agents/code-developer.md`
   - ревью: `.claude/agents/code-reviewer.md`
   - аудит безопасности: `.claude/agents/security-auditor.md`
   - поиск секретов: `.claude/agents/secret-scanner.md`
3. Выбери минимум 1–2 релевантных skills из `.claude/skills/` и следуй им.

## Правила

- Дальше действуй строго по прочитанным инструкциям, не придумывай и не пропускай шаги.
- Если данных не хватает — задай вопросы и дождись ответов пользователя.
- Всегда используй релевантных агентов из `.claude/agents/` (например, `code-developer` для разработки и `code-reviewer` для ревью).
- Всегда создавай задачи в `work/.../tasks/` для значимых работ и фиксируй прогресс в документации (`work/.../plan.md`, `docs/plans.md`, `.claude/guides/features.md`).
- Перемещай выполненные задачи в архив `work/_archive/` и оставляй активными только текущие задачи.
- Выбор skills:
  - Если задача про агентные системы (архитектура, контекст, память, инструменты, multi-agent, evaluation) — всегда подключай минимальный набор контекст-инжиниринг skills, даже если пользователь не упомянул их явно.
  - В остальных задачах подключай skills только по совпадению описаний или по прямому указанию пользователя.
