# Инструкции для ИИ

**Точка входа** для Claude Code, Codex CLI и других ИИ-инструментов.

## Быстрый старт

1. Прочитай `context.md` — стек, команды, структура
2. Если неясно — спроси и дождись ответа
3. Делай

## Правила

### Простая задача (1-3 файла)
Уточни если надо → сразу код

### Сложная задача (>3 файлов или неясна)
1. Мини-план (3-5 пунктов)
2. Дождись апрува
3. Реализуй

### Тесты
- **Обязательны**: платежи, auth, данные пользователей
- **Опционально**: CRUD, UI, утилиты

## Запрещено

- Предполагать стек без чтения `context.md`
- Выдумывать команды
- Ломать существующее

## Принципы

- **KISS** — простота
- **YAGNI** — не добавляй "на будущее"
- **DRY** — не дублируй

## Агенты (по запросу)

Специализированные агенты в `.claude/agents/`:
- `code-developer` — разработка
- `code-reviewer` — ревью
- `security-auditor` — безопасность
- `secret-scanner` — секреты

## Команды

- `/init-project` — инициализация проекта
- Остальные агенты и скиллы вызываются в чате по имени

## Автообновление

После значимых изменений обнови `context.md`:
- Добавлена зависимость
- Изменена структура
- Новые команды

## Rules Sync

Источник правил: `rules/lite.md`
Синхронизируется в: `CLAUDE.md`, `AGENTS.md`, `.cursorrules`, `.antigravity/rules.md`
